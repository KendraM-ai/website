## AP Essay Types — Quick Reference

### Synthesis Essay
- Read 6–7 sources, choose at least 3
- Build an argument and **cite sources** (Source A, Source B, etc.)
- Time: ~40 minutes

### Rhetorical Analysis
- Analyze how an author builds an argument using rhetoric
- Focus on **rhetorical strategies** (ethos, pathos, logos, syntax, diction, etc.)
- Time: ~40 minutes

### Argument Essay
- Develop your **own** evidence-based argument on a given topic
- No sources provided — use your knowledge and reasoning
- Time: ~40 minutes
